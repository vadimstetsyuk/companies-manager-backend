# companies-manager-backend

|      URL           |    Method     |            Title        |
| ------------------ | ------------- | ----------------------- |
| api/companies      |      GET      | Get all companies       |
| api/companies      |      POST     | Push company            |
| api/companies/:id  |      PUT      | Update company with :id |
| api/companies/:id  |     DELETE    | Delete company with :id |
